Introduction

For this assignment, I was asked to spend 24 hours only using a mobile browser to document my experience with different mobile sites.

Questions to consider:

  • Can you still see what you want to see?
  • Can you find what you're looking for?
  • Can you do what you want to do?
  • Any frustrations?
  • Anything work better on the small screen?

Findings

Can you still see what you want to see?

No, I was able to see what I wanted to see an almost every site. Some mobile sites have less info then desktop sites.

Can you find what you're looking for?

I was always able to find what I wanted when mobile browsing.

Can you do what you want to do?

With mobile browsing I did not have as much fredom as with desktop browsing. On desktop browsing, I am able to see more at once and do more at once.

Any frustrations?

Yes. Some drop down scroll tabs did not work properly. Most the time it opens up an ios scroll wheel on the iphone. On gamedebate.com, some of their scroll wheels on mobile did not work properly.

Anything work better on the small screen?

Some sites loaded a lot faster on mobile than on desktop because of the simplicity.

Summary

After doing this 24hr test, I was able to notice that some mobile sites are really good and some are really poor. The mobile compatibility of sites is pretty broad, just like normal desktop sites. Some sites will perform better than others.
